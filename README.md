# MeusInsumos

Sistema web para rastreamento de insumos em entidades governamentais estaduais. Acompanhe o ciclo completo de cada insumo — da emissão da ATA até a conclusão do fornecimento — com histórico auditável, lançamento de notas fiscais e integração com plataformas externas.

---

## O Problema

Entidades governamentais controlam insumos com planilhas, e-mails e sistemas isolados que não se comunicam. O resultado: nenhuma visão unificada do processo, auditoria difícil e trabalho manual excessivo.

O InsumoGov centraliza tudo isso em um único lugar.

---

## Funcionalidades

- Cadastro e gerenciamento de membros da entidade
- Rastreamento do ciclo completo: ATA → fornecimento → conclusão
- Lançamento e consulta de notas fiscais
- Histórico auditável de todas as movimentações
- Integração com plataformas de terceiros

---

## Tecnologias

- **Backend:** Python · FastAPI
- **Frontend:** Next.js (React)
- **Banco de dados:** PostgreSQL
- **Autenticação:** JWT / OAuth2
